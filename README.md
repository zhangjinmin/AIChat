> # 🤖 AI 智能分析与 DAX 助手 (Power BI Custom Visual)
>
> > 一个能让 Power BI 报表"开口说话"的自定义视觉对象。深度集成大语言模型（LLM），支持自然语言问数、智能图表生成、报表双向联动筛选、列级数据护盾，还能当你的 DAX 编写专家。
>
> 
>
> ## ✨ 核心特性
>
> ### 📊 1. 动态数据洞察与局部图表渲染
>
> * **所见即所得**：AI 读取的是**当前页面切片器筛选后的数据快照**，结论 100% 贴合你正在看的业务场景。
> * **多维动态聚合面板**：支持在前台一键切换分析颗粒度（如从“门店”上卷到“部门”）。底层 JS 引擎会在本地光速聚合数据后再发送给 AI，既节省 Token，又保证宏观分析不跑偏。
> * **智能图表生成**：基于数据特征自动匹配并渲染图表，支持：横向条形图（长文本克星）、雷达图（多维诊断）、饼图/圆环图、面积图、双轴组合图。
> * **大盘真理注入**：支持接入 DAX 计算的全局总计文本，彻底消灭 LLM 的“数学加总幻觉”。
>
> ### 🔗 2. 分析结果可交互（反向双向联动）
>
> * 当 AI 发现异常点（如某门店客流骤降、某品类流血清仓），会在报告正文中生成类似 `🔍 联动分析: 品类` 的实体按钮。
> * **一键穿透**：点击按钮，自动反向触发 Power BI 底层的 Selection Manager，让报表页内的其他所有视觉对象同步过滤，实现“AI 发现病灶 → 人工一键下钻核实”的完美闭环。
>
> ### 🛡️ 3. 数据护盾 (4 档全局隐私 + 绝对黑名单)
>
> 专为高度敏感的数据设计，支持前台实时切换，及后台强管控：
>
> | 全局隐私级别      | 说明                                    | 适用场景                                                     |
> | :---------------- | :-------------------------------------- | :----------------------------------------------------------- |
> | 🟢 **关闭 (明文)** | 原始数据透传                            | 内网测试、脱敏测试库                                         |
> | 🟡 **关键隐私**    | 门店/业务员等实体编码，保留品类业务属性 | 日常业务分析，兼顾安全与 AI 行业理解                         |
> | 🟠 **全文脱敏**    | 所有文本维度强制编码转换                | 对外协作、跨部门分享                                         |
> | 🔴 **严格隐私**    | 文本编码 + **数值动态标量混淆**         | 高敏数据。数值被随机标量缩放，但**不改变相对关系**，AI 算出的占比/排名/增幅 100% 准确。 |
>
> > 💣 **黑名单强管控 (Blacklist Override)**：可将极度敏感的字段（如 VIP 客户名、底线成本价）单独拖入“隐私黑名单”槽位。**一旦拖入，无视前台全局开关**，强制执行文本脱敏与数值混淆！
>
> ### 💻 4. 零幻觉 DAX 专家 (DAX Copilot)
>
> * 告别 AI 凭空捏造表名的报错噩梦。
> * 支持导入通过 Tabular Editor / DAX Studio 提取的**真实模型结构字典**。
> * AI 严格遵循你的物理表名、字段格式以及 `USERELATIONSHIP`（表间关系），生成复制即用的企业级代码。
>

> ## 🛠️ 安装与配置指南
>
> ### 1. 部署与导入
>
> * 确保拥有 Power BI Desktop。
> * 从 Release 下载 `.pbiviz` 文件并导入 Power BI 画布，或通过 PBI 视觉对象市场安装（待上架）。
>
> ### 2. 认识强大的“数据槽位 (Data Roles)”
>
> 本视觉对象将数据进行了极度精细化的拆分，你需要将对应的字段拖入正确的槽位中：
>
> * 🏷️ **分析维度 (Dimensions)**：拖入文本/分类/时间字段（如：门店、分类、年份）。前台会生成对应的聚合控制面板。
> * 🔢 **分析指标 (Metrics)**：拖入需要分析的数值度量值（如：销售额、客流量）。
> * 🚫 **隐私黑名单 (Privacy Blacklist)**：拖入需要**最高级别保密**的维度或指标。
> * 📜 **大盘总计播报 (Grand Total Text)**：拖入一段由 DAX 拼接的文本度量值，用于告诉 AI 大盘的真实汇总情况，防止幻觉。
> * 🤖 **系统提示词 (System Prompt)**：拖入分析 SOP 或研判规则，对 AI 实施专业洗脑。
> * ⚙️ **全局大模型配置 (Global Config)**：拖入一个包含 API BaseURL、Key 的 JSON 字符串度量值，实现统一配置下发。
>
> ### 3. 配置 API 与启动
>
> **推荐使用【配置下发法】（一改全改，无需逐个面板设置）：**
> 新建度量值 `AI_全局配置`，并将其拖入【全局大模型配置】槽位：
>
> ```dax
> AI_全局配置 = 
> "{
>   ""baseUrl"": ""[https://api.your-llm-provider.com/v1](https://api.your-llm-provider.com/v1)"",
>   ""apiKey"": ""sk-xxxxxx"",
>   ""modelName"": ""qwen-long"" 
> }"
> ```
