> > 让你的 Power BI 报表“开口说话”。不仅能自然语言问数、自动画图、帮你写 DAX，甚至点一下 AI 给出的结论，还能反向联动筛选你的整个报表！

>## 💡 这是个什么？
> 
>在 Power BI 里做完报表，业务人员依然不知道怎么看数据？
> 这是一个直接嵌入到 Power BI 画布中的 AI 分析插件。它不是脱离业务瞎聊的通用大模型，而是**贴着你当前报表的真实数据**进行深度诊断的业务外脑。
> 
> 适合零售、快消、连锁等需要频繁做品类/门店/单品下钻分析、52 周 MD 企划的实战场景。
> 
>---
> 
>## ✨ 核心大招
> 
> ### 📊 1. 懂你当前业务的“活数据”分析
>* **所见即所得**：你用切片器选了“华东区”，AI 看到的就是华东区的数据快照。
> * **治好大模型的算数病**：支持直接把 DAX 算好的大盘总计（如真实毛利率、客单价）通过专属槽位喂给 AI，彻底消灭 AI 加总错误产生的“数学幻觉”。
>* **动态聚合 vs 明细直传**：想看大盘？前端帮你秒级聚合提效；想看 52 周单品淘汰？后台一键开启“明细直传”，一行不漏让 AI 盘到底。
> 
>### 🔗 2. 结论可反向穿透联动 (这才是真 BI)
> * 当 AI 诊断出异常（例如：`⚠️ 发现【常温纯奶】流血清仓严重`）时，它会在正文里吐出一个交互按钮。
> * 你点一下这个按钮，**整个 Power BI 页面的其他图表会自动联动筛选到【常温纯奶】**，直接实现“AI 发现病灶 → 人工下钻核实”的闭环体验！
> 
> ### 📈 3. 自动匹配图表出图
> * 能用图说话的绝不用文字。AI 会根据数据特征自动渲染合适的图表：雷达图（多维诊断）、横向条形图（长文本 SKU 排名）、双轴组合图（量价齐观）等，无需写一行配置代码。
> 
>### 🛡️ 4. 懂规矩的数据护盾 (安全拉满)
> 怕商业机密被传给 API？做了护盾机制：
>* **四档全局脱敏**：从明文直传，到关键实体替换，再到最狠的**“数值动态混淆”**（数值全变，但占比、增幅的相对关系不变，AI 照样能算出对的趋势）。
> * **黑名单**：把极端敏感的列（如成本价）单独拖入黑名单槽位，无视任何开关，强行就地打码。
>
> ### 👨‍💻 5. 零幻觉的 DAX Copilot
> * 天天查 DAX 函数表？直接把 Tabular Editor / DAX Studio 导出的模型字典贴给它。
> * 它会乖乖按你真实的表名、列名和 `USERELATIONSHIP` 关系生成 DAX 代码，拿来即用，告别“AI 瞎编表名”的报错噩梦。

> ## 🛠️ 怎么用？(极简上手)
>
> ### 1. 拖拽数据，对号入座
>视觉对象提供了精细的【数据槽位】，各司其职：
> * 🏷️ **分析维度** & 🔢 **分析指标**：拖入你想让 AI 看到的基础数据表列。
> * 🚫 **隐私黑名单**：拖入绝密字段。
>* 📜 **大盘总计播报**：用 DAX 拼一段文本拖进来，防 AI 算错总账。
> * 🤖 **系统提示词**：把分析 SOP 变成文本度量值拖进来，给 AI 洗脑。
>
> ### 2. 统一配置 LLM (一键下发)
>不需要在每个图表的格式面板里痛苦配置。建一个 DAX 度量值，拖进【全局大模型配置】槽位即可全报表生效：
> 
> ```dax
> AI_全局配置 = 
> "{
>   ""baseUrl"": ""[https://api.deepseek.com/v1](https://api.deepseek.com/v1)"", // 支持任何兼容 OpenAI /v1 的接口
>   ""apiKey"": ""sk-xxxxxx"",
>  ""modelName"": ""deepseek-chat"" 
> }"



## 数据槽位 (Data Roles)

这些槽位位于 Power BI 字段面板，决定了视觉对象能“吃”进去什么数据。

| **槽位名称**       | **类型**         | **说明**                                                     |
| ------------------ | ---------------- | ------------------------------------------------------------ |
| **分析维度**       | 维度 (Grouping)  | 放入分类、日期、门店等字段。AI 会基于此在本地进行动态分组聚合。 |
| **分析指标**       | 度量值 (Measure) | 放入销售额、毛利等数值字段。这是 AI 计算与统计的数据基础。   |
| **隐私黑名单**     | 混合             | 极度敏感字段专用（如客户名、成本价）。拖入后，无视全局脱敏设定，强制脱敏/混淆。 |
| **系统提示词**     | 度量值           | 放入包含分析框架、SOP 的文本度量值，对 AI 进行“专业洗脑”。   |
| **快捷指令**       | 度量值           | 生成一键式按钮。度量值名称为按钮名，内容为 AI 执行的 Prompt。 |
| **界面说明**       | 度量值           | 用于展示在聊天框首行的引导语（如“数据更新至 2026-05”）。     |
| **大盘总计播报**   | 度量值           | **核心槽位**。拖入 DAX 计算的全局汇总文本，彻底解决 AI 计算大盘数据的幻觉。 |
| **全局大模型配置** | 度量值           | 拖入包含 `baseUrl`, `apiKey`, `modelName` 的 JSON 字符串，实现多报表统一管理。 |

## 格式设置 (Format Settings)

这些设置位于 Power BI 右侧的“格式”画笔面板。

### 1. 大模型配置 (AI Config)

- **API Base URL**: AI 接口地址（如 DeepSeek/Qwen 接口地址）。
- **API Key**: 你的模型调用秘钥。
- **模型名称**: 指定使用的模型（如 `deepseek-chat`）。

### 2. 界面设置 (UI Config)

- **AI 助手名称**: 定义聊天框头部的显示名。
- **主题颜色**: 视觉对象的主色调（按钮、气泡等）。
- **显示自动洞察按钮**: 开启后，底部显示“自动洞察”快捷键。
- **自动洞察按钮名称/提示词**: 设置一键触发的内容。

### 3. AI 模块开关 (Module Settings)

- **启用 数据洞察 模块**: 控制聊天 Tab 是否可见。
- **启用 DAX 助手 模块**: 控制 DAX 编写 Tab 是否可见。
- **启用 显示流转明细**: **开发者模式**。开启后可查看 Agent 路由逻辑、原始数据条数与聚合压缩率。
- **默认数据隐私模式**: 全局生效的脱敏强度（off/semi/full/strict）。
- **允许前台调整聚合维度**: 开启后，用户可在前端勾选维度；关闭后，报表维度被锁定，防止小白乱动。
- **禁用本地数据聚合 (直传明细)**: **MD 企划专用**。开启后跳过聚合，直接将所有底层明细推送给 AI。
- **启用 闲聊防浪导护栏**: 开启后，AI 会自动拒答所有非业务相关的问题。
